# Vibe-Trading 自然语言使用指南

> 把市场问题用大白话说出来，AI 自动帮你调用数据、跑回测、写报告。

---

## 一、基础用法

启动后直接在聊天框输入问题，就像跟一个懂金融的分析师聊天：

```
帮我分析一下 NVDA 最近 30 天的走势
```

```
回测一下 BTC-USDT 的 20 日/50 日均线金叉策略，时间范围 2024 全年
```

```
000001.SZ 目前的估值水平怎么样？跟同行业比有什么优势？
```

**关键原则**：说出来你想要什么结论、什么时间范围、什么品种，越具体结果越好。不用记参数，用自然语言就行。

## 二、市场研究

### 单票深度分析

```
帮我研究一下 600519 贵州茅台：
1. 最近两年的营收和净利润趋势
2. 当前 PE 和历史分位数
3. 机构评级和目标价
4. 跟五粮液、泸州老窖对比估值
```

```
保利发展 600048 的基本面分析：
- 2025 年报的核心财务数据
- 土地储备变化
- 股息率变化趋势
- 当前 PB 和 NAV 折价
```

### 板块/行业研究

```
A 股新能源板块目前有哪些值得关注的公司？帮我从估值、成长性、技术面三个维度筛选
```

```
房地产板块最近有什么政策变化？对龙头公司有什么影响？用数据说明
```

### 宏观分析

```
分析一下当前美联储利率路径对 A 股和黄金的影响
```

```
中美利差变化对人民币汇率和北向资金的影响有多大？
```

### 加密货币

```
BTC 现在链上数据怎么看？交易所余额、矿工流出、资金费率一起看
```

```
ETH-USDT 的永续合约资金费率最近一周的变化趋势，有没有套利机会？
```

## 三、策略回测

### 技术指标策略

```
用 20 日和 60 日均线金叉死叉策略回测 000300.SH（沪深 300），2019 到 2024 年，
输出年化收益、最大回撤、夏普比率
```

```
测试一下 BTC-USDT 的布林带突破策略：
价格突破上轨做多，跌破下轨做空，用最近两年的数据
```

```
帮我回测一个 RSI 均值回归策略：
上证 50 成分股中，当 14 日 RSI 低于 30 时买入，高于 70 时卖出，
持股周期 5-20 天，最大回撤控制在 10% 以内
```

### 多因子策略

```
用沪深 300 成分股回测一个动量+低波+质量的复合因子策略：
选过去 6 个月涨幅前 20% 且波动率后 50% 且 ROE 前 30% 的股票，
月度调仓，回测 2018 到 2025
```

### 回测后续操作

回测跑完后可以：

```
帮我导出这个策略的 Pine Script 代码，我要放到 TradingView 上看
```

```
这个策略的夏普比率只有 0.8，帮我想想怎么改进？是不是加一个止损条件会好一点？
```

## 四、团队分析（Swarm）⭐ 核心功能

启动一个虚拟的分析师团队，多人从不同角度辩论后给出结论。

### 4.1 投资委员会 — 多空辩论

```
启动投资委员会，帮我辩论一下：特斯拉现在到底该不该买？
让多方和空方各自陈述理由，风控组评估风险后由 PM 做最终决策
```

```
用 investment_committee 分析英伟达：
正方从 AI 算力需求增长、数据中心资本开支角度论证
反方从估值泡沫、竞争加剧、出口管制角度反驳
最后给我一个综合建议
```

### 4.2 财报分析团队

```
启动财报研究团队，分析苹果最新的季报：
1. 基本面组梳理营收结构、利润率变化、现金流
2. 分析师预期组对比实际数据 vs 市场一致预期
3. 期权组分析财报后的隐含波动率变化
最后给我一个综合报告
```

### 4.3 量化策略团队

```
用量化策略团队帮我做一个完整的因子研究流程：
1. 从 A 股市场中筛选候选 alpha 因子
2. 回测验证 IC、IR、多空收益
3. 风险审计组检查过拟合和幸存者偏差
最终给出可以实盘的因子组合建议
```

### 4.4 宏观/外汇团队

```
启动宏观利率外汇团队，分析：
日本央行加息对日元套利交易的影响，
以及对新兴市场债券和美国科技股的传导路径
```

### 4.5 技术分析委员会

```
用技术分析委员会分析黄金走势：
请经典技术指标派、一目均衡表派、谐波形态派、艾略特波浪派各抒己见，
最后给出综合判断和具体的支撑/阻力位
```

### 4.6 风险委员会

```
启动风险委员会，审查我的持仓组合：
我现在持有 40% A 股、30% 美股、20% 黄金、10% 现金，
帮我做一次压力测试，看看在以下场景下的最大回撤：
1. 中美贸易战升级
2. 美联储意外加息 50bp
3. A 股流动性危机
```

### 4.7 加密货币交易台

```
用 crypto_trading_desk 分析一下当前的加密市场：
1. 资金费率和基差交易机会
2. 链上鲸鱼流入流出
3. 稳定币流动方向
4. 清算热力图
最后风控经理给一个风险敞口建议
```

### 4.8 全球资产配置委员会

```
启动全球配置委员会，帮我做下半年的资产配置方案：
我目前 100 万人民币，风险偏好中等偏保守，
请分别给出 A 股、港股、美股、加密货币、债券的配置比例和理由
```

### 4.9 事件驱动团队

```
启动事件驱动团队，分析以下事件的投资机会：
美国总统签署的 AI 芯片出口管制新规对半导体供应链的影响，
哪些公司受益？哪些受损？有什么套利机会？
```

### 4.10 行业轮动团队

```
用 sector_rotation_team 分析 A 股当前的行业轮动信号：
1. 哪些行业处于动量强势阶段？
2. 哪些行业出现反转信号？
3. 结合宏观经济周期，下一阶段应该超配哪些行业？
```

### 团队功能的自然语言技巧

好的 prompt：
```
启动投资委员会，帮我做一次全面的宁德时代投资分析，参考最新季报和市场数据
```

不好的 prompt：
```
帮我看宁德时代
```

区别在于：**明确说要启动团队（swarm/委员会/团队/委员会），给出分析维度，让 AI 知道需要多个角色协作**。

## 五、Alpha Zoo（因子库）

让 AI 从 452 个预建因子中挑选适合你策略的因子。

### 浏览和验证

```
帮我列出 gtja191 因子库中跟反转相关的因子，我想找适合做均值回归的因子
```

```
对比一下 gtja191_171、gtja191_111、gtja191_163 这三个因子在沪深 300 上的表现，
帮我看哪个 IC 最高、IR 最好，给出排序和建议
```

```
用 alpha101 因子库，在沪深 300 成分股上做一次基准测试，
帮我找出 IC 均值最高且存活率超过 60% 的前 10 个因子
```

### 组合策略

```
从 gtja191 因子库中选 5 个低相关性的因子组成多因子策略：
1. 要求 IC 均值 > 0.03
2. 因子间相关性 < 0.3
3. 在沪深 300 上回测 2018-2025
```

## 六、交易日志分析（Shadow Account）

上传你的券商交易记录，让 AI 分析你的交易行为。

```
我上传了去年的交割单，帮我：
1. 总结我的交易风格和偏好
2. 找出我的行为偏差（处置效应、过度交易、追涨杀跌等）
3. 提取我的隐含交易规则，生成一个"影子策略"
4. 回测影子策略，跟我实际交易对比，看如果严格执行规则能多赚多少
```

## 七、跨会话记忆

让 Vibe-Trading 记住你的偏好，下次自动调用。

```
记住：我喜欢 RSI 类策略，最大回撤不超过 10%，持股周期 5-20 天，
行业偏好消费和科技，不碰周期股
```

下次直接说：
```
帮我找几个符合我偏好的策略机会
```

AI 会自动读取之前的偏好。你也可以随时查看和删除记忆：
```
帮我看看你记住了我哪些偏好
忘记我之前关于周期股的限制
```

## 九、实战场景组合

### 场景一：A 股投资者日常

```
1. 先帮我看看今天 A 股大盘走势和北向资金流向
2. 帮我分析 000858 五粮液最近的财务公告和机构评级变化
3. 回测一下白酒板块的低估值+高股息策略
```

### 场景二：加密货币交易员

```
1. 用 crypto_trading_desk 分析当前市场情绪和资金流向
2. 回测 ETH 的动量突破策略，最近 6 个月
3. 看看 BTC 矿工余额和交易所流入流出，判断短期方向
```

### 场景三：研究员写报告

```
帮我写一份关于中国新能源车产业链的研究报告：
1. 启动全球配置委员会，给出行业配置建议
2. 用量化策略团队筛选关键因子，做归因分析
3. 用宏观利率团队分析政策影响
4. 最终生成一份包含数据图表的结构化报告
```

### 场景四：自省交易行为

```
1. 上传我去年的所有交易记录
2. 让 AI 分析我的行为模式，找出亏钱的原因
3. 提取我的交易规则，生成影子策略
4. 对比影子策略和实际交易，看差距在哪
5. 记住我最容易犯的错误，下次分析时提醒我
```

## 十、小技巧

| 技巧 | 例子 |
|------|------|
| **越具体越好** | "回测 2023-2024 的 20 日/60 日均线策略"比"回测均线策略"好 |
| **告诉 AI 你要什么格式** | "用表格列出近三年营收、净利、毛利率变化" |
| **先保存偏好** | 说一次"记住我偏好…"，后面省很多反复描述 |
| **不满意就追问** | "这个回测的最大回撤太大了，帮我加一个 5% 止损" |
| **跨轮次继续** | "刚才的分析我还不满意，帮我加上行业对比" |
| **善用团队** | 复杂决策→启动团队，简单查询→直接问 |
| **让 AI 自我纠错** | "检查一下你刚才的回测有没有未来函数，数据范围是否合理" |
