# Vibe-Trading 因子投资完全指南（小白版）

> 你只有投资想法，不懂代码、不懂金融术语。这份指南教你怎么用大白话，从想法到实盘信号，全程不用写一行代码。

---

## 开场：你只需要知道这些

**因子是什么？** 就是一个打分规则。比如"缩量回踩均线"这个想法，翻译成因子就是：

> 给每只股票打分 —— 趋势越强（在均线之上）、回踩越到位（接近均线）、量缩得越明显，分数越高。分数高的股票就是你要关注的。

Vibe-Trading 内置了 452 个专业因子，你不需要自己写公式，只需要用自然语言找到跟你想法最接近的因子，让它帮你验证和组合。

## 完整流程（以"缩量回踩均线"为例）

整个流程分 6 步，每一步都是在聊天框里说人话：

```
想法 → 找因子 → 验证因子 → 组合策略 → 回测验证 → 导出实盘信号
```

---

### 第 1 步：描述你的想法

直接告诉 Vibe-Trading 你的思路，越具体越好：

```
我有一个选股想法：找上升趋势中缩量回踩 20 日均线的股票。

具体条件：
1. 股价在 20 日和 60 日均线之上（上升趋势确认）
2. 最近一周内出现过一次回踩到 20 日均线附近（偏离不超过 3%）
3. 回踩那天的成交量明显萎缩（低于前 5 天平均成交量的 70%）
4. 排除 ST 股和市值小于 50 亿的股票

帮我看看 Alpha Zoo 里有没有跟这个想法接近的因子？
```

Vibe-Trading 会搜索 452 个因子库，告诉你哪些因子跟你的想法最接近。

---

### 第 2 步：验证因子是否有效

找到了候选因子（比如 `gtja191_004`），别急着用，先验证：

```
帮我验证 gtja191_004 这个因子在 A 股上到底有没有效：

1. 在沪深 300 成分股上跑 IC 测试，时间范围 2019-2025
2. 告诉我 IC 均值、IR、IC 为正的比例、存活率
3. 分成上涨市、下跌市、震荡市分别看表现
4. 跟单纯持有沪深 300 指数对比
```

> **怎么判断因子好不好？**
> - IC 均值 > 0.03 → 不错，> 0.05 → 优秀
> - IR > 0.5 → 稳定，> 1.0 → 很稳定
> - IC 为正比例 > 55% → 方向一致性好

如果因子表现不理想：

```
gtja191_004 的 IC 只有 0.02，太低了。帮我找找 gtja191 库里跟它相关性低但 IC 更高的因子，我想组合使用
```

---

### 第 3 步：对比多个因子

不要只依赖一个因子，多看几个：

```
帮我对比这三个因子在沪深 300 上的表现：
- gtja191_004（量价突破）
- alpha101_030（动量+成交量收缩）
- gtja191_048（动量+量比）

时间范围 2019-2025，按月 IC 排序，输出对比表格
```

Vibe-Trading 会给出每个因子的 IC、IR、胜率等数据，帮你选。

---

### 第 4 步：组合成策略

挑好因子后，让 AI 帮你组合：

```
帮我用 gtja191_004 和 alpha101_030 做一个复合选股策略：

1. 两个因子标准化后等权相加，得到综合得分
2. 在沪深 300 成分股里，每月月初选综合得分前 20 名的股票
3. 加入风控：
   - 单只股票仓位不超过 10%
   - 跌破 60 日均线就止损
   - 行业集中度不超过 30%
   - 最大回撤超过 15% 时空仓
4. 回测时间 2019 到 2025 年，月度调仓
```

---

### 第 5 步：回测验证策略

策略写好后，让它跑回测：

```
帮我把上面的策略跑完整回测，输出：
1. 年化收益率、年化波动率、夏普比率、最大回撤
2. 每年收益拆分（2019-2025 每年表现）
3. 跟沪深 300 指数的对比（超额收益、跟踪误差、信息比率）
4. 月度胜率、平均盈亏比
5. 最大连续亏损月份数
6. 如果效果不好，请帮我分析原因并给出改进建议
```

如果回测结果不错（年化跑赢指数 5%+，夏普 > 0.8），就可以进入下一步。

如果不好：

```
夏普比率只有 0.5，最大回撤 35%，太差了。
帮我想想怎么改进：
- 是不是该加入基本面因子过滤（ROE、毛利率）？
- 要不要限制只在低波动市场环境下使用？
- 加一个趋势过滤（大盘在 60 日线上方才开仓）会好一点吗？
```

---

### 第 6 步：导出实盘信号

回测通过后，生成可以直接用的东西：

```
帮我把这个策略导出成：
1. TradingView 的 Pine Script 指标（我可以直接在图上看到信号）
2. 每周自动筛选的股票池（满足条件的股票列表）
```

然后你可以：

1. **在 TradingView 上用**：把 Pine Script 代码粘贴进去，图形化看到买卖信号
2. **每周跑一次**：`vibe-trading run -p "用我的策略筛选本周符合条件的股票"`

## 选股策略常用的自然语言模板

你可以把下面这些当成"填空题"，填入你的想法直接用：

### 趋势类

```
帮我找股价在 [20日/60日/120日] 均线之上，且均线 [多头排列/向上发散] 的股票，
排除 [ST股/市值小于X亿/涨停板]
```

### 回调类

```
找 [上升趋势中/创新高后] 回调到 [20日/60日] 均线附近的股票，
回调幅度不超过 [3%/5%/8%]，
成交量 [萎缩/缩小到前5天均量的60%以下]
```

### 突破类

```
找股价 [突破/站上] [20日/60日/年线] 的股票，
突破当天成交量 [放大到前5天均量的1.5倍/放量明显]，
[之前经历过一段整理/底部盘整超过1个月]
```

### 基本面+技术面

```
帮我筛选：
1. ROE > [15%]，毛利率 > [30%]
2. 股价在 60 日线上方
3. 最近一个月涨幅 < [20%]（不要追高）
4. 市盈率 < 行业平均
```

### 资金面

```
找我关注的股票里：
北向资金最近 [5天/10天] 持续净买入的，
且股价 [横盘整理/小幅回调] 还没大涨的
```

## 附录一：专业名词大白话解释（小孩都能听懂版）

> 你不需要背这些术语。Vibe-Trading 会帮你算好、解读好。但这个附录帮你"翻译"一下 AI 说话时蹦出的那些词是什么意思。

---

### IC（信息系数 / Information Coefficient）

**一句话：** 因子打分准不准。

**类比：** 假设你给班里的同学预估期末考试成绩。考试结束后，你的预估排名和实际排名越接近，你的"预测能力"就越强。IC 就是衡量这个"预测能力"的数字。

- IC = 1 → 你完全猜对了，100 分
- IC = 0.5 → 猜对了一半
- IC = 0 → 瞎蒙的水平
- IC < 0 → 你说涨的它跌了，反向指标

**判断标准：**
| IC 值 | 好不好 | 比喻 |
|--------|--------|------|
| > 0.05 | 优秀 | 学霸的答案 |
| 0.03~0.05 | 不错 | 靠谱的参考 |
| < 0.03 | 一般 | 天气预报（有时对有时错） |
| < 0 | 反向 | 反着来可能更赚 |

---

### IR（信息比率 / Information Ratio）

**一句话：** 因子的预测能力稳不稳定。

**类比：** 两个射箭选手 — 小明平均射 8 环，有时候 10 环有时候 2 环；小刚平均射 7 环，但每次都是 6-8 环之间。小刚的 IR 比小明高。IR 高 = 每次预测都差不多靠谱，不会忽好忽坏。

**判断标准：**
| IR 值 | 稳不稳 |
|--------|--------|
| > 1.0 | 很稳定，放心用 |
| 0.5~1.0 | 还可以，能用 |
| < 0.5 | 不太稳，最好跟其他因子搭配 |

---

### IC 为正比例 / 胜率

**一句话：** 在所有测试月份里，因子猜对方向的比例。

**类比：** 一年 12 个月，如果因子在 8 个月里都预测对了涨跌方向，胜率就是 8/12 = 67%。胜率越高，说明这个因子不是靠运气。

- > 60% → 不是蒙的
- < 50% → 还不如抛硬币

---

### 夏普比率（Sharpe Ratio）

**一句话：** 你每承担一单位风险，能赚多少钱。

**类比：** 两条路到同一个目的地 — 一条是平坦高速（波动小），一条是九曲山路（波动大）。虽然都到了 100 公里外的地方，但你愿意走哪条？夏普比率就是告诉你：同样赚 10%，你是"平稳赚到的"还是"坐过山车赚到的"。

**判断标准：**
| 夏普比率 | 好不好 | 比喻 |
|----------|--------|------|
| > 1.5 | 优秀 | 坐高铁 |
| 1.0~1.5 | 不错 | 开高速 |
| 0.5~1.0 | 一般 | 走国道 |
| < 0.5 | 差 | 骑自行车上高速 |
| < 0 | 亏钱 | 走着去更快 |

---

### 最大回撤（Max Drawdown）

**一句话：** 你的账户从最高点跌到最低点，最多亏了多少。

**类比：** 你往存钱罐里放钱，最高的时候里面存了 100 块。后来因为买了玩具，最低只剩 60 块。那你的"最大回撤"就是 40%。如果你最多只能接受亏 20%，那就得在这之前收手。

**判断标准：**
- < 10% → 风险很低（保守型）
- 10%~20% → 风险适中
- 20%~30% → 风险偏高
- > 30% → 波动很大，心脏不好的别玩

---

### 年化收益率（Annual Return）

**一句话：** 平均下来每年赚多少钱。

**类比：** 你有 100 块本金，5 年后变成 161 块。年化收益不是 61/5=12.2%，而是 10%。因为每年的收益也在"利滚利"—— 第一年赚 10 块变 110，第二年赚 11 块变 121……

- A 股长期平均年化约 8~10%
- 如果策略能稳定做到 15%+，已经是高手水平

---

### 年化波动率（Volatility）

**一句话：** 你的账户一年到头上下晃得有多厉害。

**类比：** 坐飞机 — 气流稳定的时候你喝咖啡一点不洒（低波动），遇到气流颠簸咖啡洒一裤子（高波动）。波动率就是衡量这种"颠簸程度"的数字。波动率 20% = 你的账户一年大概会在 ±20% 的范围内晃来晃去。

---

### 超额收益 / Alpha

**一句话：** 你的策略比大盘指数多赚的部分。

**类比：** 全班平均分 70 分，你考了 85 分。那你的"超额"就是 15 分。如果大盘今年涨了 8%，你的策略赚了 13%，那你的 alpha 就是 5%。Alpha 是衡量你"真本事"的指标 — 赚的多可能只是大盘好，但 alpha 高说明你真的比别人会选股。

---

### Beta

**一句话：** 你的策略跟大盘有多"像"。

**类比：** 你坐一条小船，前面有一条大船拖着走。大船（大盘）往东走，你也往东走。但是加大油门的小船（高 beta）大船走 1 米它能走 1.5 米；拖不动的重船（低 beta）大船走 1 米它只走 0.5 米。

- Beta = 1 → 大盘涨 1%，你也涨 1%
- Beta > 1 → 大盘涨 1%，你涨超过 1%（进攻型）
- Beta < 1 → 大盘涨 1%，你涨不到 1%（防守型）

---

### 因子暴露 / Factor Exposure

**一句话：** 你的策略押注在某个"特征"上有多少。

**类比：** 做菜 — 这道菜里盐放了多少？酱油放了多少？"因子暴露"就是在问：你的策略里有多少成分是靠"动量"赚钱的？有多少是靠"价值"赚钱的？太多鸡蛋放同一个篮子就不安全。

---

### 过拟合（Overfitting）

**一句话：** 拿着答案做题，当然全对，但换套卷子就完蛋。

**类比：** 你背下了所有考试真题的答案，考试时全部答对。但换套新题，你就不会了。策略也一样 — 如果参数调了太多、规则太复杂，回测好看但实盘崩盘，就是过拟合了。

**记住：规则越简单，越不容易过拟合。**

---

### 存活率 / Survivorship

**一句话：** 因子在所有时间里"活着"（有效）的比例。

**类比：** 一个游戏角色，玩了 100 个小时，其中 70 小时在输出伤害，30 小时在躺地板。存活率就是 70%。因子也一样 — 有些年在赚钱，有些年在亏钱，存活率告诉你它"在线"的时间比例。

## 常见问题

### Q: 回测结果很好，实盘会一样吗？

不一定。回测是历史数据，实盘有滑点、流动性、市场变化等因素。建议：

- 回测时间越长越好（至少覆盖一轮牛熊，3-5 年）
- 策略规则越简单越好（参数越多越容易过拟合）
- 把回测结果打 7 折当实盘预期
- 实盘先小仓位跑 3 个月验证

### Q: 我不会写 Python，能行吗？

全程不需要写代码。本文所有带引号的例子，直接复制粘贴到 Vibe-Trading 聊天框就能用。

### Q: 多久跑一次？

- 月度策略：每月的第一个交易日跑一次
- 周度策略：每周日晚上跑一次
- 日内：不需要，Vibe-Trading 目前主打中长期策略
